Basic Formatting using Inline CSS

CSS, or Cascading Style Sheets, is a language used for describing the presentation of a web page written in HTML or XML. It allows web developers and designers to separate content (HTML) from layout and design, making it easier to maintain and update websites. CSS controls everything from font sizes, colors, and spacing to complex layouts, animations, and responsiveness for different screen sizes.
